### 右移后或运算

```java
    public static int function(int n){
        n|=n>>1;
        n|=n>>2;
        n|=n>>4;
        n|=n>>8;
        n|=n>>16;
        return n;
    }
```

这个只要看 n 的二进制最高位在哪里，经过上述程序后就会将最高位的 1 一直复制到最低位。

### 例子

```java
n=100; // 0110 0100
```

经过上述程序后，变为：

```java
n=127; // 0111 1111
```

###  一句话总结

最高位的 1 会复制传递到最后一位。
